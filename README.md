## Dependencies

[flex/bison](https://www.gnu.org/software/bison/) - for parsing epddl

[expect](http://manpages.ubuntu.com/manpages/trusty/man1/expect.1.html) - for timeout

## Compile and Run

change to the directory `MEPK/src`

```
$ make
$ ./run.sh
```
